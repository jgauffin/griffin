Griffin.Converter Namespace
Example Send Feedback
Converter namespace is used to create an abstraction between converters, mappers and your code.

Interfaces

  InterfaceDescription
IConverter<(Of <(<'TFrom, TTo>)>)>
Use to convert from a specific type to another.
IConverterProvider
A provider is used to provide a certain kind of conversion to the layer.
IConverterService
Framework used to convert between different object types.

Remarks

The purpose is not only to get a layer that can do simple conversions like string to DateTime, but to be able to provide conversions between different objects like an domain model to a view model.

Look at the Griffin.Converter.AutoMapper project to see how external libraries can be integrated into the converter layer.

Examples

CopyC#
// use the automapper converter.
converterService.Register(new AutoMapperConverter<EntityUser, UserViewModel>());

// use it for more complex conversions
var viewUsers = converterService.ConvertAll<EntityUser, UserViewModel>(users);

// or simple ones.
var myDate = converterService.Convert<string, DateTime>("2011-02-10");
