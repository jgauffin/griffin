Remarks

Log entries which is helpful during debugging. The amount of debug entries can be vast, and it's therefore not recommended to have them turned on in production systems unless it's really required.

Examples

CopyC#
public void MyClass
{
    public ILogger _logger = LogManager.GetLogger<MyClass>();

    public string CalculateQuestion(int answer)
    {
        _logger.Debug("Answered {0}.", answer);
        return "So you want to know the real question, ehhh?";
    }
}
